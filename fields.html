WHAT IS FRONTEND?

(Data attribute)

Frontend is the digital representation of a company, individual, or organization that builds a bridge between a customer or client and the brand1. It is everything with which the user interacts, including the user interface and the interface design and programming that makes the interface function2345. The frontend is responsible for the total look and feel of an online experience and is essential to user interactions and visual journeys45.

HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It defines the structure and content of web content. When you create a web page, you use HTML elements to enclose different parts of the content, making it appear and behave in specific ways.

CSS

(Contenteditable)

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of web documents written in HTML or XML. It allows you to control how elements are rendered on screen, paper, speech, or other media. With CSS, you can style text, layout, and even create effects like animation.

JAVA SCRIPT

(Title attribute)

JavaScript is a cross-platform, object-oriented scripting language used to make webpages interactive. It adds dynamic functionality and complex features like interactivity and animation to web pages. When a web page does more than just display static information—such as showing timely content updates, interactive maps, animated graphics, or scrolling video jukeboxes—you can bet that JavaScript is probably involved12. It’s the third layer of the standard web technologies stack, with HTML defining the structure of web content and CSS handling styling. JavaScript enables you to create dynamically updating content, control multimedia, and perform a wide range of tasks with just a few lines.
